一、实验目的、实验原理及实验基本要求
 1. 明确本次实验的学习目标与实操要求
2. 讲解 ESP32 开发环境基本工作原理
3. 梳理整个实验的操作流程与考核要点
 
二、清点实验所用硬件设备、软件工具并完成前期准备
 1. 逐一检查 ESP32 开发板、数据线等硬件器材
2. 准备 Arduino IDE 安装包、串口驱动等软件资源
3. 检查计算机网络、USB 接口等外部使用环境
 三、完成 Arduino IDE 开发软件的下载、安装与个性化设置
 1. 从官方渠道获取适配 Windows 系统的安装程序
2. 按照安装向导完成软件主体安装流程
3. 对编辑器界面、行号、字体等功能进行自定义配置
4. 试运行软件，确认基础功能正常可用
 四、配置开发板源地址、安装驱动与 ESP32 开发支持库
 1. 在软件偏好设置中添加 ESP32 开发板官方链接
2. 下载并安装 CH340/CP210x 串口驱动程序
3. 通过开发板管理器在线安装 ESP32 全套支持文件
4. 检查驱动安装结果，确认设备被系统正常识别
五、对 ESP32 开发板各项运行参数进行完整配置
1. 选定对应的 ESP32 开发板型号
2. 设置上传波特率、闪存频率、工作模式等核心参数
3. 根据硬件配置选择分区方案与附加功能选项
4. 选择计算机与开发板对应的通信串口
六、硬件接线、设备检查与整体实验平台调试
1. 使用专用数据线连接 ESP32 开发板与计算机
2. 观察开发板电源指示灯状态，排查硬件连接问题
3. 再次核对软件参数配置，确保软硬件匹配一致
七、编写测试程序并完成代码编译、调试与烧录
1. 新建空白工程，编写板载 LED 闪烁测试代码
2. 执行代码编译操作，排查并修正语法错误
3，将调试完成的程序烧录至 ESP32 开发板中
八、观察实验现象、读取串口数据并完成结果记录
1. 观察板载 LED 亮灭状态，记录闪烁规律
2. 开启串口监视器，设置对应波特率参数
3. 查看串口打印信息，记录输出内容与格式
4. 对比理论预期与实际现象，完成结果记录
